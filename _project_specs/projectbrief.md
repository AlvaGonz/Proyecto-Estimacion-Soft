# EstimaPro - Project Brief

## Project Identity
- **Name:** EstimaPro (Plataforma Web Multi-Metodo para la Estimacion Colaborativa de Software)
- **Type:** University Capstone Project
- **Institution:** Universidad Central del Este (UCE), Escuela de Ingenieria de Software
- **Year:** 2024
- **Authors:** Adrian Alexander Alvarez Gonzalez | Ray Ruben Ventura Lopez
- **Director:** Ing. Julio Alexis
- **Docente:** Ing. Francisco Santana

## Core Purpose
A web platform that digitizes, automates, and optimizes collaborative software estimation processes. The system enables Facilitators to select from multiple industry-recognized estimation techniques (Wideband Delphi, Planning Poker, and Three-Point Estimation), adapting the estimation experience to each teams methodological needs.

## Primary Goals
1. Implement a configurable estimation engine supporting at least three methods
2. Develop user and role management system (Administrator, Facilitator, Expert)
3. Create project estimation management module
4. Implement iterative estimation rounds mechanism
5. Develop statistical calculation engine
6. Create interactive graphical visualizations
7. Implement asynchronous communication system
8. Develop convergence detection algorithm
9. Create report generation module
10. Guarantee security and privacy
11. Design responsive user interface
12. Develop historical action log
13. Design adaptive UI for estimation process

## Scope Boundaries
**In Scope:**
- Full analysis, design, development, implementation, and validation of estimation platform
- Three estimation methods: Wideband Delphi, Planning Poker, Three-Point (PERT)
- Role-based access control (RBAC)
- Statistical analysis and visualization
- Report generation (PDF/Excel)
- Docker containerized deployment

**Out of Scope (Initial Version):**
- External tool integrations (Jira, Trello, Azure DevOps)
- Native mobile applications
- AI/ML prediction algorithms
- Integrated video conferencing
- Post-estimation resource management
- Billing/financial management
- Multi-language support (Spanish only initially)
