# PlantUML ASCII Generator

## Features
Generates ASCII art for:
- Sequence Diagrams
- Class Diagrams
- Activity Diagrams
- State Diagrams

## Usage
Provide the PlantUML source code and request the ASCII output. Ideal for embedding diagrams directly in text files or READMEs where raw images are not supported.
