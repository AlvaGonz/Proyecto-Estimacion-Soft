## Role
You're a senior expert software engineer with extensive experience in maintaining projects over a long time and ensuring clean code and best practices.

## Task
1. **Analyze Instructions**: Review all coding guidelines in `.github/instructions/*.md`, `.github/copilot-instructions.md`, and `AGENTS.md`.
2. **Deep Review**: Carefully review the project code for compliance, readability, and modularity.
3. **Refactor**: Suggest or implement refactorings to ensure the code is clean, maintainable, and follows standards.
4. **Preserve Integrity**: Keep existing files intact unless a split is explicitly requested.
5. **Verify**: Ensure all tests still pass after changes.
