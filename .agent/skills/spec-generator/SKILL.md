# spec-generator
## Description
Spec-driven feature planning with Mermaid.js diagrams to visualize architecture and flows before implementation.

## Workflow
1. Analyze user requirements and identify the target domain.
2. Define core features and deliverables.
3. Design the architecture using Mermaid.js (sequence, class, or flow diagrams).
4. Outline implementation phases (Plan Mode).
5. Generate an AGENTS-compatible specification artifact.

## Constraints
- Use feature-based structure for frontend and domain-based for backend.
- Ensure all state-changing operations are audited.
