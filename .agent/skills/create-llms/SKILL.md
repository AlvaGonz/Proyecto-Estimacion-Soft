# LLMs.txt Generator

## Directive
Generate an `llms.txt` file following the official specification to help AI agents understand the repository structure, purpose, and key files.

## Components
1. **Summary**: High-level purpose of the repo.
2. **Structure**: Mapping of key directories and files.
3. **Guidelines**: Coding standards or architectural constraints.
4. **Links**: Pointers to documentation or external resources.

## Best Practices
- Keep it concise.
- Use standard section headers.
- Update whenever significant architectural changes occur.
